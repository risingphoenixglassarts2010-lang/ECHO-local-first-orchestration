# FAQ

## What is ECHO?
A local-first cognitive architecture for safely mediating LLM intent.

## Does ECHO act autonomously?
No.
ECHO amplifies human agency — it does not pursue goals.

## Why seven layers?
Each layer isolates a cognitive responsibility to maintain clarity and safety.

## Is the internal logic open-source?
No.
Only the architecture and philosophy are public.

## Can ECHO run offline?
Yes — local-first is a core design principle.

## Does ECHO replace human judgment?
Never.
It strengthens it.
