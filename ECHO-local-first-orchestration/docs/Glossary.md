# ECHO Glossary

**Doctrine** — The philosophical core that defines identity and values.  
**Interpreter** — The stabilizer that ensures coherent behavior.  
**State** — Memory compression and continuity scaffolding.  
**HUD** — Cognitive overlay that highlights what matters.  
**Execution** — Deterministic workflows and structured action.  
**Integrity** — Safety, rollback, and coherence protection.  
**Evolution** — Long-term refinement and meta-alignment.  

This glossary helps new readers understand ECHO’s terminology.
