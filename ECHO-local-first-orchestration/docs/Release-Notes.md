# Release Notes — v0.1 Initial Architecture Release

This release introduces the full public structure of the ECHO architecture.

## Included
- Seven-layer architecture folders
- Preface (philosophy)
- Full documentation suite
- Text-based architecture diagram
- Runtime model
- Glossary
- Design principles
- Roadmap
- FAQ

## Not Included
- Internal logic
- Doctrine text
- Interpreter prompts
- Workflow internals
- State structures

This release establishes the foundation for future refinement.
